# SPRINT_7_PROYECT
Proyecto del Sprint 7 Tripleten
El objetivo es desarrollar una aplicación web interactiva con Streamlit que permita explorar y visualizar datos mediante gráficos interactivos de Plotly, dando interacción con el usuario mediante botones y casillas de verificación, y desplegarla en la nube usando Render, con todo el proyecto versionado en GitHub.”
